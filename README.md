# Full Stack Reference Materials

This repository is created for **referring to full stack development learning materials**.

It contains notes, sample code, examples, and practice files for different technologies used in full stack development.

The main goal of this repository is to:
- Store learning resources in one place
- Provide easy reference for concepts
- Help with revision and practice

This repository includes materials related to:

- Frontend frameworks and libraries
- Backend development
- Databases
- APIs
- Version control (Git)
- Deployment basics

## How to Use This Repository

You can use this repo to:
- Review concepts
- Practice coding examples
- Prepare for interviews
- Build mini projects

This repository is only for **learning and reference purposes**.
